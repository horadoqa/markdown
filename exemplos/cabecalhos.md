# Cabeçalhos

Cabeçalhos são usados para organizar e estruturar o conteúdo em seções.
No Markdown, eles são definidos utilizando o símbolo `#`.

A quantidade de `#` indica o nível do cabeçalho (de `h1` até `h6`).

---

## Sintaxe

Código:

```markdown id="8v2n3k"
# Este é um cabeçalho h1
## Este é um cabeçalho h2
### Este é um cabeçalho h3
#### Este é um cabeçalho h4
##### Este é um cabeçalho h5
###### Este é um cabeçalho h6
```

Resultado:

# Este é um cabeçalho h1

## Este é um cabeçalho h2

### Este é um cabeçalho h3

#### Este é um cabeçalho h4

##### Este é um cabeçalho h5

###### Este é um cabeçalho h6

---

## Boas práticas

* Use apenas um `h1` por documento (título principal)
* Utilize os níveis de forma hierárquica (não pule de `h1` para `h4`, por exemplo)
* Mantenha os títulos curtos e descritivos
* Evite excesso de níveis para não dificultar a leitura

---

## Dica

Você pode adicionar uma linha em branco antes e depois dos cabeçalhos para melhorar a legibilidade do código Markdown.
