# Markdown 

Markdown é uma linguagem de marcação leve utilizada para formatar texto de maneira simples e rápida, sem a necessidade de ferramentas complexas, como editores avançados.

A proposta do Markdown é permitir que você escreva usando símbolos fáceis de lembrar (como #, *, -), que podem ser convertidos em HTML ou já exibidos formatados em diversas plataformas, como GitHub, Notion e vários editores de texto.

## Por que usar Markdown?
- Simples e fácil de aprender
- Leve e rápido de escrever
- Amplamente suportado em diferentes plataformas
- Ideal para documentação, anotações e conteúdos web
