# Alertas

Alertas são uma extensão Markdown baseada na sintaxe blockquote que você pode usar para enfatizar informações críticas. Eles são exibidos com cores e ícones distintos para indicar a importância do conteúdo.

Código:
```markdown
> [!NOTE]
> Informações úteis que os usuários devem saber, mesmo ao apenas passar os olhos pelo conteúdo.
```

Resultado:
> [!NOTE]
> Informações úteis que os usuários devem saber, mesmo ao apenas passar os olhos pelo conteúdo.

Código:
```markdown
> [!TIP]
> Dicas úteis para fazer as coisas melhor ou com mais facilidade.
```

Resultado:
> [!TIP]
> Dicas úteis para fazer as coisas melhor ou com mais facilidade.

Código:
```markdown
> [!IMPORTANT]
> Informações essenciais que os usuários precisam saber para alcançar seu objetivo.
```

Resultado:
> [!IMPORTANT]
> Informações essenciais que os usuários precisam saber para alcançar seu objetivo.

Código:
```markdown
> [!WARNING]
> Informações urgentes que exigem atenção imediata do usuário para evitar problemas.
```

Resultado:
> [!WARNING]
> Informações urgentes que exigem atenção imediata do usuário para evitar problemas.

Código:
```markdown
> [!CAUTION]
> Alerta sobre riscos ou possíveis consequências negativas de determinadas ações.
```

Resultado:
> [!CAUTION]
> Alerta sobre riscos ou possíveis consequências negativas de determinadas ações.
