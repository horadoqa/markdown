# Menu com âncoras

## Como funciona:

* O Markdown transforma automaticamente os títulos (`##`) em **âncoras**.
* O link `[Instalação](#instalação)` aponta para o título `## Instalação`.
* Em geral, os espaços viram hífen (`-`) e letras ficam em minúsculo.

```markdown
## Menu

- [Introdução](#introdução)
- [Instalação](#instalação)
- [Uso](#uso)
- [Exemplos](#exemplos)
- [Conclusão](#conclusão)

---

## Introdução
Aqui você coloca a introdução do seu conteúdo.

## Instalação
Passos para instalar o projeto ou ferramenta.

## Uso
Explicação de como utilizar.

## Exemplos
Alguns exemplos práticos de uso.

## Conclusão
Considerações finais.
```

Resultado:

## Menu

- [Introdução](#introdução)
- [Instalação](#instalação)
- [Uso](#uso)
- [Exemplos](#exemplos)
- [Conclusão](#conclusão)

---
