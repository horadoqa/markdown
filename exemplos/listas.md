# Listas

Listas são utilizadas para organizar informações de forma clara e hierárquica.
No Markdown, você pode criar listas não ordenadas, ordenadas e até listas de tarefas.

---

## Não ordenada

Listas não ordenadas utilizam símbolos como `*`, `-` ou `+`.

Código:

```markdown
* Item 1
* Item 2
  * Item 2a
  * Item 2b
    * Item 3a
    * Item 3b
```

Resultado:

* Item 1
* Item 2

  * Item 2a
  * Item 2b

    * Item 3a
    * Item 3b

---

## Ordenada

Listas ordenadas utilizam números seguidos de ponto (`1.`).

Código:

```markdown
1. Item 1
2. Item 2
3. Item 3
   1. Item 3a
   2. Item 3b
```

Resultado:

1. Item 1
2. Item 2
3. Item 3

   1. Item 3a
   2. Item 3b

---

## Listas com formatação

Você pode combinar listas com outros elementos de formatação, como itálico e negrito.

Código:

```markdown
1. Item 1
2. Item 2
   * _Item 2a em itálico_
   * **Item 2b em negrito**
3. Item 3
```

Resultado:

1. Item 1
2. Item 2

   * *Item 2a em itálico*
   * **Item 2b em negrito**
3. Item 3

---

## Lista de Tarefas

Listas de tarefas (checklists) são úteis para acompanhar atividades.
Elas são suportadas em plataformas como GitHub e Notion.

Código:

```markdown
- [x] Tarefa concluída
- [ ] Tarefa pendente
```

Resultado:

* [x] Tarefa concluída
* [ ] Tarefa pendente
